    Act as user {{ course.code }}

    The sudo option allows you to act as if you are another user.

    However, you will only be able to act as that user within this course, {{ course.code }}. Additionally, you will only be able to sudo as one user across all courses: enabling the sudo option in another course will disable it for the current course.

    An indicator in the navigation bar will display the user for which you have activated the sudo option. It will also indicate whether the option is activated for the current page with either (Sudo enabled for this page) or (Sudo disabled for this page) .

    You can return to this page at any time, by clicking on the indicator. To cancel the sudo option, select the empty option from the dropdown below.
